# VHE-dataset
Dataset for PACLIC 2024 paper ["VHE: A New Dataset for Event Extraction from Vietnamese Historical Texts"](https://aclanthology.org/2024.paclic-1.59/)

## Overview
VHE is the Vietnamese dataset for event extraction and event relation extraction. It contains 4,114 instances with 5,213 events and 7,423 event arguments that pertain to historical events from the Hong Bang dynasty (2879 BC) to the Later Le dynasty in the sixteenth century.
